This swift caller is only for macs. 

To use it. 

Download the swift caller.

Double click to open the zip file. Inside is a file called swift.app
Drag and drop this into your applications folder.\
Now when you go into launch pad if you search for swift it will turn
up - click on it and it will start. So if your speaker is connected to a
bluetooth speaker it will play from there.

The swift.app will only play:

In the morning between 6:30 and 11:30

In the evening between 16:00 and 20:00

If the weather forecast for Hassocks according to
[[https://open-meteo.com]{.underline}](https://open-meteo.com) 
currently gives the chance of rain as less than 50%.
